# Repositório de Estudos de Matemática utilizando Python

## Sobre o repositório

Este repositório foi criado com o objetivo de documentar minha jornada acadêmica no curso de Bacharelado em Ciência e Tecnologia (BC&T) pela UFABC, com a intenção de me especializar (formar) em Matemática.

Aqui organizo meus estudos de matemática de forma estruturada, utilizando Python como ferramenta de apoio para experimentação, simulação e consolidação dos conceitos aprendidos ao longo da graduação.

---

## Objetivos

* Revisar e consolidar fundamentos da matemática acadêmica
* Desenvolver raciocínio lógico e pensamento formal
* Traduzir conceitos matemáticos em código Python
* Criar pequenos projetos para aplicar teoria na prática
* Construir uma base sólida para áreas como Data Science e Inteligência Artificial

---

## Organização

* Cada pasta de conteúdo contém:

  * `tema-resumo.md`: resumo teórico e anotações
  * `exercicios.py`: prática dos conceitos estudados
